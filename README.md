# STAXX

**Stack Smarter.**

The gamified DeFi simulator and crypto sim-trading platform built on Solana. Learn DeFi by playing — risk-free — and earn real $STAXX tokens.

## Features

- **Sim Trade** — Trade BTC, ETH, SOL, LTC with live market data and multi-timeframe charts
- **Perpetuals** — Long/Short with up to 100x leverage
- **DeFi Simulator** — Practice real DeFi strategies with simulated protocols and random market events
- **Staking** — Stake $STAXX for 12.5% APY with auto-compound
- **Tournaments** — 10-minute trading sprints with $STAXX prizes
- **Token Search** — Look up any Solana token by contract address via DexScreener
- **Phantom Wallet** — Connect, view balances, withdraw/deposit $STAXX
- **Quests** — Complete challenges, earn XP, stack $STAXX

## Tech Stack

- React 18 + Vite
- Live price data: CoinGecko, CoinPaprika, Binance
- Token search: DexScreener API
- Wallet: Phantom (Solana)
- Smart contract: Anchor framework (Rust)
- Hosting: Vercel

## Getting Started
